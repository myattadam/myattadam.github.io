# HTML to PDF converter tool

- [PhantomJS](http://phantomjs.org/) - HTML to PDF (Open Source)
- [wkhtmltopdf](https://wkhtmltopdf.org/) - HTML to PDF (Open Source)
- [Prince](http://princexml.com/) - HTML to PDF (Commercial, Server License)
- [PDFreactor](http://www.pdfreactor.com/) - HTML to PDF (Commercial, Server License)
- [DocRaptor](https://docraptor.com/) - HTML to PDF (Commercial, SaaS API)
- [jsPDF](https://github.com/MrRio/jsPDF) - PDF “programming” library (Open Source)

> **Prince** seems to be the best option at the moment
